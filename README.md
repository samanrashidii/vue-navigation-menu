## Getting Started

Clone the project from Github:

```
git clone git@github.com:samanrashidii/vue-navigation-menu.git
```

Install and save dependencies:

```
npm install --save --verbose
```

Start the project:

```
npm start
```

Build project:

```
npm run build
```
